How to Save Changes

  1. Click "Generate JSON" to see your current blog data
  2. Copy the entire JSON output below
  3. Open blog-data.json in your text editor
  4. Replace all content with the copied JSON
  5. Save the file
  6. Refresh your blog page to see the changes
